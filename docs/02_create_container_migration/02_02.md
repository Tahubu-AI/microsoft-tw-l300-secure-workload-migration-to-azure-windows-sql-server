# Task 02 - Deploy the Container Migration Solution Accelerator

## Introduction

In this Task, the goal is to deploy the Container Migration Solution Accelerator locally and validate the deployment success.

## Description

After you complete this task, you will be able to:

- Deploy the accelerator locally 
- Validate deployment success 

## Success Criteria

* Successfully validate that the deployment of the Container Migration Solution Accelerator is successful

## Solution

<details markdown="block">
<summary>Expand this section to view the solution</summary>

1. From the Hyper-V VM that you have been working with in the previous steps, you need to make sure 2 more installs take place AZD and Pythong 3.9+

2. First install Azure Developer CLI `azd` from a powershell instance:

```powershell
    powershell -ex AllSigned -c "Invoke-RestMethod 'https://aka.ms/install-azd.ps1' | Invoke-Expression"
```

![AZD install](../../Hands-on%20lab/images/02-02-azd-install.png)

3. Verify azd was installed successfully.  Remember you need to restart your Powershell instance

```powershell
    azd version
```

![AZD verify](../../Hands-on%20lab/images/02-02-azd-verify.png)

4. Install Python on the Hyper-V VM, version 3.9 or above.  You can download it from this [link](https://www.python.org/downloads/)

5. After completing the install of the flavor of Python above 3.9 that you chose, verify that Python is installed successfully on the Hyper-V VM:

```powershell
    python --version
```

![Python verify](../../Hands-on%20lab/images/02-02-python-verify.png)

6. Next you need to authenticate with Azure Developer CLI (azd), use the following command

```powershell
    azd auth login
```

Login with your own credentials

7. Change directory in the powershell instance to where you cloned the `Container Migration Solution Accelerator` repo

8. Execute the following command in the powershell instance to start the provisioning process

```powershell
    azd up
```

9. Choose a short name for the environment name, for example `conmig`, your subscription that has enough quote for o3 AI model and choose your region.  This step take between 4 to 6 minutes to complete.

![azd up](../../Hands-on%20lab/images/02-02-azd-up.png)

10. Verify the deployment was successful by logging into the Azure Portal and confirming the resources and the deployment success.

![Deployment Success](../../Hands-on%20lab/images/02-02-deployment-success.png)


</details>
