# Exercise 02 - Create a Container Migration

## Lab Scenario

In this lab, attendees will perform steps toward migrating TailSpin containerized workloads to Azure Kubernetes Services (AKS) using the Container Migration Solution Accelerator. TailSpin needs to prepare its environment, deploy the accelerator, migrate workloads, configure post-migration settings, and validate business outcomes.  

## Objectives

After you complete this lab, you will be able to:

* Prepare the environment for Container Migration.
* Deploy the Container Migration Solution Accelerator.
* Understand the business use case, outcomes, and architecture.
* Perform container migration and final configuration.
* Leverage advanced capabilities of the accelerator.

## Lab Duration

* **Estimated Time:** 50 minutes
